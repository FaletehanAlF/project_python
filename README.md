# To-Do List Python

Program sederhana To-Do List menggunakan bahasa Python.  
Program ini dapat digunakan untuk menambahkan, melihat, dan menghapus tugas melalui terminal.

## Fitur

- Tambah tugas
- Lihat daftar tugas
- Hapus tugas
- Keluar dari program

## Struktur Menu

```text
1. Tambah Tugas
2. Lihat Tugas
3. Hapus Tugas
4. Keluar
